## 📖 About The Project

This project is a high-fidelity front-end clone of the Razorpay landing page, built from scratch to master complex layouts, CSS transforms, and responsive design. It utilizes a lean, framework-free environment to focus heavily on core front-end structural skills, custom geometries, and modern utility-first styling.

## ✨ Key Features

* **Complex UI Geometry:** Accurate reproduction of Razorpay's signature skew-transformed sections, absolute positioning, and multi-layered backgrounds.
* **Smooth Animations:** CSS-driven hover effects, infinite scrolling company logos, and interactive feature cards.
* **Modern Utility Styling:** Fully styled using the latest utility classes for rapid and scalable UI development.
* **Blazing Fast Build:** Leverages a modern bundler for instant server start and lightning-fast HMR (Hot Module Replacement).

## 🛠️ Built With

* **HTML5**
* **Vanilla JavaScript**
* **Tailwind CSS v4**
* **PostCSS**
* **Vite**
* **Feather Icons**

## 🚀 Getting Started

Follow these simple steps to get a local copy up and running.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/razorpay-clone.git](https://github.com/your-username/razorpay-clone.git)

2. Navigate into the project directory:
 
    cd razorpay-clone

3. Install the dependencies:
  
    npm install

4. Start the development server:
  
    npm run dev
